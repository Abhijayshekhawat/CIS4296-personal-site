Student Feedback Form

The project is about creating a web application that allows for student feedback collection. It will be a survey form for gathering feedback on a course and its professor. The form will contain different controls for inputting data and a server-side page to process the results, display them, and calculate grades based on the feedback.

Homebuilder

HomeBuilder is a web application project aimed at allowing home buyers to design their dream homes online. This involves choosing from a variety of home plans and customizing features such as roofing materials, countertops, flooring, and additional home amenities. The application will manage a database to store the selections and sales information, and provide dynamic data displays. The user interface includes controls for inputting buyer information and selections, with server-side processing for validation and calculation of total costs. The project emphasizes a professional, user-friendly design and efficient data management.

Dating Site

This Project focuses on the development of an online dating site, enhancing skills in multi-tier web application construction, database interaction, and data modeling. This project enables members to create detailed profiles, search for potential matches based on shared interests and preferences, and communicate with others through likes and date requests. A significant emphasis is placed on privacy, with certain information being kept private until mutual interest is established. The design and functionality aim to be both professional and user-friendly, incorporating advanced features such as dynamic content display, account and profile management, and the implementation of stored procedures for database operations.
